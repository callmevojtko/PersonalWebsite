MLB Bet Emailer

A simple machine learning project that uses Random Forest Regressor to predict the winning spread of baseball games based on sportsbook lines. The project fetches data from The Odds API, preprocesses it using pandas, trains a model using scikit-learn, and sends an email with the day's recommended bets using the Gmail API.
